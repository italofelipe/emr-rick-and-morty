### EMR - Rick and Morty

Como executar:

- Clone o projeto, e com o terminal, execute `npm install` ou `yarn`
- Após a instalação, para rodar o projeto, execute `npm run dev`. Isso executará o projeto localmente.

### Tech Stack:

Core:
- Next.js
- Styled Components
- Apollo Client

### Development tools:
- Jest
- @testing-library/react
- Cypress
- Storybook
- GraphQL Codegen
- ESLint

### Git hooks:
- Husky
- Lint Staged
- Commitlint

### CI/CD
- GitHub Actions
- Cypress Cloud
- SonarCloud
- Chromatic
